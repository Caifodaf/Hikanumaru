# HikanumaruApp
Hicanumaru app - it is an application for reading manga, ranobe, and much more. 

# Стек
```
– Kotlin
– One Activity
– Retrofit2, Moshi
– REST API 
– Coil
– Corutines, Flow
– MVVM, Hilt, Dagger2
– Room, SharedPreferences
– Livedata, ViewBinding
– Figma, Postman
```
